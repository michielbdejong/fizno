FIZNO: the physical knowledge box

A fizno is a little box, the size of a book, that emits a wifi signal. Through this, you can access all kinds of knowledge! From reference material like Wikipedia, OpenStreetMap, and StackOverflow, to educational resources like Khan Academy and codecademy.

I want one! How can I get a fizno?

At this point, fizno.com is only a hackathon project, sorry! But you if you're a bit handy with linnux, you can build your own fizno on an old laptop or ona plug server, using the Docker image.
